**Replace with:**

```cpp
#pragma once

// EIDAuthenticateVersionText MUST remain as #define - used in .rc resource files

```

**Rationale:** These macros are used by the Visual Studio resource compiler (RC.exe) which cannot process C++ constexpr or const expressions. The resource compiler only understands preprocessor #define macros.
